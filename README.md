# [Open Dialogue Multimodal](https://open-dialogue-multimodal.vercel.app/)

テキスト/音声入出力機能を搭載したオープンダイアローグチャットボットアプリver.2です。
デバイスやブラウザにより音声入出力に制限があります。（要復旧）
